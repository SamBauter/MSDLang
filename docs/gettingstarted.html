Getting Started

Downloading and Building the Executable

  1. You will need to download all of the files contained in the github folder found here
  2. If you don't already have cmake installed then you will need to download it. Instructions for installing on various platforms can be found here
  3. In your terminal application navigate to the folder containing the files dowloaded from the github folder.
  4. Run make
  5. This should generate an two files:
    File 1: msdscript: an executable file for running msdscript in the command line.
    File 2: libmsdscript.a: a library file enabling the User to link the classes, functions, and methods of MSDLang to produce their own executable.
    Note: Using these files is discussed in the: User guide.
  6. Please proceed to the Grammar Guide to learn about the grammar supported by MSDScript.
