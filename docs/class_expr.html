virtual bool Expr::equals ( std::shared_ptr< Expr >  e)
pure virtual

Checks the equality of two expr objects without interpretation.

Note: Order matters! Ex: 1+2 != 2+1. Note: Expressions that simplify or interpret to the same result are not equal.

virtual void Expr::step_interp ( )
pure virtual

A variation of the interp method which allocates heap memory to support Exprs that require large amounts of recursion to interpret.

Does not return a value but sets a global variable Step::val to the result. This value can be returned by the interp_by_steps method of the Step class
